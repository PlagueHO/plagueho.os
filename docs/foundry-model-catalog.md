# Microsoft Foundry Model Catalog

> **Last updated:** 2026-03-17
> **Total models:** 116
> **Publishers:** Anthropic, Black Forest Labs, Cohere, DeepSeek, Meta, Microsoft, Microsoft (Azure OpenAI), Mistral AI, Moonshot AI, NTT Data, Nixtla, Stability AI, xAI

## Summary

This catalog snapshot, taken on 2026-03-17, lists **116 models** available in the Microsoft Azure AI Foundry model catalog across **13 publishers**. It includes models from Azure OpenAI (GPT-5.x, GPT-4.x, o-series), Anthropic (Claude), Meta (Llama 4 / Llama 3.x), Mistral AI, Cohere, DeepSeek, xAI (Grok), Moonshot AI (Kimi), Black Forest Labs (FLUX), Stability AI, Nixtla, and NTT Data. Models are available via serverless (pay-per-token) deployments in select Azure regions, or as Global Standard deployments available in all Azure regions. Managed compute deployments are also available for many open-weight models.
