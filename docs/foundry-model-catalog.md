Regional availability data for Azure AI Foundry models requires authenticated access to the Azure ML REST API (`management.azure.com`) and is not available through public data sources. To check regional availability for a specific model, use the [Azure AI Foundry portal](https://ai.azure.com) or the Azure CLI:

```bash
az ml model list --registry-name azureml --query "[].{name:name,version:version}"
```
